Server which listens for game state updates from Dota 2.

The Dota client must be configured to send updates to the server. Create a text file like gamestate_integration_test.cfg in the folder:

dota 2 beta/game/dota/cfg/gamestate_integration/
Content copied to clipboard

With contents like:

"My GSI test"
{
    "uri"           "http://localhost:44444"
    "timeout"       "5.0"
    "buffer"        "0.1"
    "throttle"      "0.1"
    "heartbeat"     "30.0"
    "data"
    {
        "provider"      "1"
        "map"           "1"
        "player"        "1"
        "hero"          "1"
        "abilities"     "1"
        "items"         "1"
    }
}

Note that the port (44444) can be changed to any valid port.
